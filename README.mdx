# Terraform AWS Practice

This repository contains my Terraform exercises while learning Infrastructure as Code and preparing for the **AWS Certified Developer – Associate (DVA)** certification.

The goal of this repository is to practice provisioning and managing AWS infrastructure using Terraform and to better understand how different AWS services work together in real-world scenarios.

## Technologies

* Terraform
* AWS EC2
* AWS Application Load Balancer
* AWS Auto Scaling Group
* AWS Security Groups
* ...

## Purpose

* Practice Infrastructure as Code (IaC)
* Improve Terraform skills
* Build AWS infrastructure using code
* Prepare for the AWS DVA certification exam
